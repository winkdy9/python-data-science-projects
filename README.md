# Python Data Science Projects

Набор учебно-исследовательских проектов по анализу данных и машинному обучению на Python.

## О проекте

Репозиторий содержит практические реализации методов обработки, анализа и визуализации данных, а также эксперименты с алгоритмами машинного обучения.

Основная цель проекта — изучение полного цикла работы с данными: от подготовки и исследования датасетов до построения моделей и оценки их качества.

## Основные направления

### 1. Exploratory Data Analysis (EDA)

Исследование и анализ данных с помощью статистических методов и визуализации.

В рамках проектов выполняются:

- загрузка и обработка данных;
- очистка и подготовка датасетов;
- анализ распределений;
- поиск зависимостей между признаками;
- визуальное представление результатов.

---

### 2. Data Visualization

Изучение методов представления данных для выявления закономерностей и скрытых зависимостей.

Используемые подходы:

- построение графиков;
- анализ корреляций;
- сравнение характеристик данных;
- визуальный анализ результатов.

---

### 3. Machine Learning

Реализация и исследование алгоритмов машинного обучения.

Основные этапы:

- подготовка обучающих данных;
- разделение выборок;
- обучение моделей;
- проверка качества;
- анализ полученных результатов.

Рассматриваются базовые подходы машинного обучения и особенности их применения для решения практических задач.

---
